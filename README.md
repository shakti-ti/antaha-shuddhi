# Yoga Studio SPA (Boilerplate)

Single-page React + Vite starter for a Yoga studio website. Content is intentionally minimal placeholders.

## Start the dev server

```bash
npm install
npm run dev
```

## Build for production

```bash
npm run build
```

## Preview the production build

```bash
npm run preview
```

## Deploy to GitHub Pages

This repo is configured to auto-deploy with GitHub Actions.

1. Push this project to a GitHub repo named `antaha-shuddhi`.
2. In GitHub, open **Settings → Pages** and set **Source** to **GitHub Actions**.
3. Push to `main` (or run the **Deploy to GitHub Pages** workflow manually).
4. Your site will be published at:
	`https://<your-github-username>.github.io/antaha-shuddhi/`
